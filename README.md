# PlayWithPDF

A step taken to maintain the privacy. This offline image to pdf convertor saves Data, time and most important <b>PRIVACY</b>. 

# Requirements

# 1 Linux
```
* Pillow
* tkinter
```
# 2. Windows

## Requirements:
```
pip/pip3 install Pillow
```
## Execution:
```
python/python3 PlayWithPDF.py
```
